# 用户

`whoami` 当前用户

`who` 登录过的所有用户

`exit` 退出当前用户

`useradd abc -m` 添加用户abc

`userdel abc`删除abc，不会删除主目录

`userdel -r abc`删除abc所有

`su`切换用户

`su root`切换为root

`sudo`获取管理员权限

`sudo -s`切换为管理员权限

# 用户组

`groupadd groupdel `添加删除组账号

`usermod -g 用户组 用户名`修改用户所在组

`chown 用户名 文件或者目录`修改文件所有者

`chgrp 用户组名 文件或目录名`修改文件所属组
